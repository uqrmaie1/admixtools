# Phase 2 (PR δ) staging fixtures

Generated during the Phase 2 LLD verification pass on 2026-05-20.
Used to ground the LLD's claims about LEGOFIT output formats and
the bootci.py pipeline against real outputs.

These files are NOT yet in any branch of the admixtools fork; they
will be vendored into the PR δ branch when work on Phase 2 starts.

## Files

- `ourex1.lgo`: a 3-leaf no-admix topology generated via
  `graph_to_lgo()` (PR γ). The xyz/xy split.
- `ourex1.opf`: site pattern probabilities generated via
  `legosim -1 -d 0 ourex1.lgo`. Used as the "real data" input to
  `legofit`.
- `ourex1.legofit`: fit produced by
  `legofit --threads 1 -1 -d 0 ourex1.lgo ourex1.opf`.
- `ourex1_boot0..4.legofit`: synthetic bootstrap replicates,
  generated by perturbing `ourex1.legofit` (NOT a real bootstrap
  pipeline — sufficient for format testing). See the LLD note on
  fixture generation in `tests/testthat/fixtures/README.md` for
  when Phase 2 lands.
- `ourex1.flat`: combined flat file via `flatfile.py`.
- `ourex1.bootci`: confidence intervals via `bootci.py`.

## Tool versions

- LEGOFIT git hash `3675373-dirty` (master as of 2026-05-18)
- Python 3 (system)

## Pipeline command

```bash
LEGOSIM=/Volumes/Bioinformation/at2Contrib/legofit/src/legosim
LEGOFIT=/Volumes/Bioinformation/at2Contrib/legofit/src/legofit
FLATFILE=/Volumes/Bioinformation/at2Contrib/legofit/src/flatfile.py
BOOTCI=/Volumes/Bioinformation/at2Contrib/legofit/src/bootci.py

$LEGOSIM -1 -d 0 ourex1.lgo > ourex1.opf
$LEGOFIT --threads 1 -1 -d 0 ourex1.lgo ourex1.opf > ourex1.legofit

# Synthetic bootstrap (real bootstrap = repeat with resampled data)
for i in 0 1 2 3 4; do
  PERTURB_XYZ=$(awk "BEGIN { print 2 + ($i - 2) * 0.05 }")
  PERTURB_XY=$(awk "BEGIN { print 0.5 + ($i - 2) * 0.02 }")
  sed -e "s/T_xyz = 2$/T_xyz = $PERTURB_XYZ/" \
      -e "s/T_xy = 0.5$/T_xy = $PERTURB_XY/" \
      ourex1.legofit > ourex1_boot${i}.legofit
done

python3 $FLATFILE ourex1.legofit ourex1_boot*.legofit > ourex1.flat
python3 $BOOTCI ourex1.flat > ourex1.bootci
```
